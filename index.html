關於 index.html

什麼是 index.html？

index.html 是一個常見的網頁文件名稱，通常用於網站的首頁。它是使用 HTML（超文本標記語言）撰寫的，並提供了網站的基本結構和內容。

用途

當用戶訪問一個網站的根目錄時，伺服器通常會自動加載 index.html 文件，這使得它成為用戶進入網站的第一個接觸點。

基本結構

一個典型的 index.html 文件包含以下幾個部分：

  • 文檔類型宣告：指定 HTML 的版本。
  • 部分：包含元數據，如標題、字符集和樣式。
  • 主體部分：顯示頁面內容，包括文本、圖片和其他元素。

示例代碼

以下是一個簡單的 index.html 示例：

<!DOCTYPE html>
<html lang="zh-Hant">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的網站</title>
</head>
<body>
    <h1>歡迎來到我的網站！</h1>
    <p>這是我的首頁內容。</p>
</body>
</html>

結語

index.html 是每個網站的核心部分，無論是個人網站還是商業網站，它都是用戶的第一印象。了解其結構和用途是學習網頁設計的重要一步。
